## 🏗️ Project Structure

```text
src/
  assets/images/    — optimized WebP image assets
  components/       — reusable UI, navigation and footer components
  layouts/          — main layout, header and footer wrappers
  modules/          — interactive behavior modules
  pages/            — page sections and page-specific components
  styles/           — Tailwind entry point and custom styles
  main.js           — application entry point
```
